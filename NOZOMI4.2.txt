NOZOMI


NOZOMI - COMPLETE DEVELOPMENT CONCEPT & TECHNICAL BLUEPRINT
PROJECT VISIE
Nozomi is een futuristische conversational Japanese learning companion gebouwd als een lichte PWA (Progressive Web App).
De focus ligt op:
	•	natuurlijke gesprekken
	•	immersive UX
	•	speech interaction
	•	guided language learning
	•	companion sfeer
	•	realtime interactie
	•	lage latency
	•	lage hardwarevereisten
	•	gratis/local-first architectuur
Nozomi moet aanvoelen als:
een intelligente digitale aanwezigheid waarmee je Japans oefent op een natuurlijke manier.
Niet als:
	•	een klassieke lesapp
	•	een AI demo
	•	een ChatGPT clone
	•	een overload aan knoppen en functies
De ervaring moet:
	•	kalm
	•	premium
	•	vloeiend
	•	futuristisch
	•	minimalistisch voelen.
 
⸻
 
CORE FUNCTIONALITEITEN
Conversational Japanese Learning
Gebruiker kan:
	•	typen
	•	praten via microfoon
	•	realtime gesprekken voeren
Nozomi:
	•	antwoordt natuurlijk
	•	toont Japans
